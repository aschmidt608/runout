Load the collection
This app can't call the Discogs API directly — it runs in a locked-down browser sandbox that blocks outbound requests to anything but Anthropic's own API. Pull your collection with a one-time local script instead; your token never leaves your machine.
  1. Get a token at Discogs → Settings → Developers → "Generate new token."
  2. Download fetch_discogs_collection.py (shared alongside this app) and run:
    python fetch_discogs_collection.py YOUR_USERNAME YOUR_TOKEN
  3. Drop the discogs_collection.json it writes into the box below.

Automatic copy isn't available here — the text below is selected, so press Cmd/Ctrl+C to copy it.
Assumptions baked in: the collection comes from a local JSON file (this sandbox can't reach the Discogs API directly, and Discogs' own CSV export leaves out genre/style), so re-run the script when your collection changes. Track duration isn't pulled per-release either, so length is estimated from the avg-minutes-per-track field above. Tempo/origin is a hint passed to Claude, not filtered against Discogs metadata directly. Tidal links open a search for each track rather than auto-adding — full push-to-playlist needs a registered OAuth app with a fixed redirect URI, which this standalone tool doesn't have.
